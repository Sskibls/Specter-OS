Privacy Features

Built from the ground up for maximum privacy and security

🎭

Persona Shards

Isolated environments for different identities - Work, Anon, Burner, and Lab. Each shard has independent storage, network routes, and policies.

🔌

Network Kill Switch

Instant network disconnect for emergencies. One keystroke blocks all network traffic to protect your identity.

🔐

Full Disk Encryption

LUKS2 encryption by default. Your data stays protected even if your device is compromised.

📋

Audit Logging

Tamper-evident security event logging. Know exactly what happened and when.

🧅

Tor Integration

Built-in Tor support for anonymous browsing. Route any application through Tor with one click.

🛡️

Secure Boot

UEFI Secure Boot support with custom key enrollment. Protect your boot chain from tampering.

🔄

A/B Updates

Seamless system updates with rollback support. Never brick your system again.

🔒

DNS over HTTPS

Encrypted DNS queries by default. Prevent DNS snooping and manipulation.
