INSTALLATION

Install SpecterOS

Complete step-by-step guide to install SpecterOS on your system

📋 Prerequisites

⚡ Processor

2 cores minimum, 4 cores recommended (x86_64)

🧠 Memory

2 GB RAM minimum, 4 GB recommended

💾 Storage

20 GB minimum, 64 GB SSD recommended

📀 Boot Media

USB drive (8GB+) or DVD

📌 Note: Back up all important data before installation. The installation process will erase data on the target disk.

📥 Step 1: Download SpecterOS

  1. Download the ISO Download the latest SpecterOS ISO from the download page. The file size is approximately 3.2 GB.
  2. Verify the checksum Verify the SHA256 checksum to ensure the download is complete and uncorrupted:
    sha256sum specteros-debian-20240305.iso
    Compare with the checksum on the download page.

💾 Step 2: Create Bootable USB

  1. Using Rufus (Windows) Download Rufus from rufus.ie, select the SpecterOS ISO, choose your USB drive, and click Start.
  2. Using Etcher (Windows/macOS/Linux) Download Balena Etcher, select the ISO file, choose your USB drive, and click Flash.
  3. Using dd (Linux/macOS) Open terminal and run:
    sudo dd if=specteros-debian-20240305.iso of=/dev/sdX bs=4M status=progress && sync
    Replace /dev/sdX with your USB drive (e.g., /dev/sdb).
⚠️ Warning: Creating a bootable USB will erase all data on the USB drive. Make sure to back up any important files first.

🚀 Step 3: Boot from USB

  1. Insert the USB drive Insert the bootable USB drive into your computer.
  2. Access boot menu Restart your computer and press the boot menu key (usually F12, F10, F2, or Del depending on your manufacturer).
  3. Select USB drive Select your USB drive from the boot menu and press Enter.
  4. Select boot option From the SpecterOS boot menu, select "SpecterOS (Live)" to try before installing, or "SpecterOS (Install)" to install directly.

🔧 Step 4: Installation Process

  1. Welcome screen Select your language and click "Continue".
  2. Keyboard layout Choose your keyboard layout and test it in the text box. Click "Continue".
  3. Installation type Choose one of the following options:
    • Erase disk and install: Recommended for most users. Will use entire disk with encryption.
    • Manual partitioning: For advanced users who want custom partition layout.
  4. Encryption setup If you chose encryption, enter a strong passphrase. This will be required at every boot.
    🔐 Security Tip: Use a long, unique passphrase (at least 16 characters). Store it in a password manager.
  5. User account Create your user account:
    • Enter your name
    • Choose a username
    • Set a strong password
    • Choose a hostname (default: specteros)
  6. Timezone Select your timezone or click on the map. Click "Continue".
  7. Installation Review your settings and click "Install". The installation will take 15-30 minutes.

✅ Step 5: First Boot

  1. Remove USB and reboot When installation completes, remove the USB drive and click "Restart Now".
  2. Enter encryption passphrase If you enabled encryption, enter your passphrase at the boot screen.
  3. Login Enter your password at the login screen to access your desktop.
  4. Welcome wizard The SpecterOS welcome wizard will guide you through initial setup including Persona Shards configuration.

🔧 Post-Installation

Update System

sudo apt update && sudo apt upgrade -y

Enable Persona Shards

Open the Shard Manager from the applications menu and configure your isolated environments.

Configure Network Privacy

Open Network settings to configure DNS over HTTPS and Tor integration.

Install Additional Software

sudo apt install package-name

❓ Troubleshooting

Boot Issues

If SpecterOS doesn't boot, try adding nomodeset to the boot parameters:

Press 'e' at boot menu → Add 'nomodeset' after 'quiet' → Press F10

Wi-Fi Not Working

Some Wi-Fi cards require proprietary firmware. Connect via Ethernet first, then run:

sudo apt install firmware-linux-nonfree

Installation Fails

If installation fails, check:

  • USB drive is properly created (try recreating)
  • ISO checksum matches
  • Sufficient disk space (20GB minimum)
  • Secure Boot is disabled in BIOS
