# PhantomKernel Prompt Pack

This folder contains the full prompt structure for building `PhantomKernel OS` with different model roles.

Files:

- `MASTER_PROMPT.md` — General high-quality full spec prompt
- `CODEX_PROMPT.md` — Implementation-focused prompt
- `KIMI_PROMPT.md` — Architecture consistency-focused prompt
- `OPUS_PROMPT.md` — Adversarial security review-focused prompt

Suggested usage:

1. Start with `MASTER_PROMPT.md` for baseline spec generation.
2. Run `CODEX_PROMPT.md` for implementation planning.
3. Run `KIMI_PROMPT.md` for architecture validation and traceability.
4. Run `OPUS_PROMPT.md` for adversarial hardening and risk review.
